# Expense Tracker

A simple React app to track expenses. Built with React and Vite.

## Features

- Add expenses with description and amount
- View list of expenses
- Delete expenses
- View total amount spent
- Responsive and clean UI

## Tech Stack

- React (with Hooks)
- Vite (build tool)
- CSS for styling
